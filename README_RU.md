# Hosts Editor

Простой и легкий инструмент командной строки для управления файлом `hosts` в Windows. Написан на Batch и PowerShell.

## Возможности

- **Просмотр (List)**: Отображение всех активных и отключенных (закомментированных) записей.
- **Добавление (Add)**: Добавление новых записей (IP и домен).
- **Удаление (Remove)**: Удаление записей по домену.
- **Переключение (Toggle)**: Включение или отключение записей без их удаления.
- **Поиск (Search)**: Поиск конкретных записей.
- **Резервное копирование (Backup & Restore)**: Безопасное создание резервных копий и восстановление.

## Установка

1. Скачайте исходный код.
2. Нажмите правой кнопкой мыши на `install.cmd` и выберите **Запуск от имени администратора**.
3. Инструмент будет установлен в системную директорию. Теперь вы можете использовать команду `hosts` в любом терминале (CMD, PowerShell).

## Использование

Откройте терминал (CMD или PowerShell) от имени **Администратора** для выполнения команд, изменяющих файл.

### Просмотр всех записей
```bash
hosts list
```

### Добавление новой записи
```bash
hosts add 127.0.0.1 example.com
hosts add 0.0.0.0 ads.example.com
```

### Включение/Отключение записи
```bash
hosts toggle example.com
```

### Удаление записи
```bash
hosts remove example.com
```

### Поиск записи
```bash
hosts search google
```

### Резервное копирование и восстановление
```bash
# Создать резервную копию
hosts backup

# Восстановить из файла резервной копии
hosts restore C:\Windows\System32\hosts_backups\hosts_backup_20230101_120000
```

## Удаление

1. Нажмите правой кнопкой мыши на `uninstall.cmd` и выберите **Запуск от имени администратора**.
2. Следуйте инструкциям для удаления инструмента и (по желанию) папки с резервными копиями.
